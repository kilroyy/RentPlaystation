Maaf Koneksi Anda Terputus / Anda Sedang Offline


Berikut langkah - langkah memastikan koneksi anda :

1. Pastikan internet tersambung / paket data tersedia.
2. Buka browser, coba untuk browsing sesuatu.
3. Jika belum bisa tutup aplikasi (force close) kemudian buka lagi.
4. Jika masih belum bisa, restart tablet.
5. Jika masih belum bisa, coba ganti koneksi ke wifi atau thetring anda.
